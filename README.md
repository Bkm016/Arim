# Arim （城市）

## 本工具库集合:
- 双栈计算器 - Saukiya
- 符号树变量计算器 - Saukiya
- 条件表达式求值器 - 枫溪
- 物品匹配工具 - 枫溪
- 生物&方块发光工具 - Gei
- 文件夹读取工具 - 枫溪

## 使用方法

1. 添加依赖

```kts
dependencies {
    taboo("top.maplex.arim:Arim:VERSION") // 替换为最新版本
}
```

2. 设置重定向

```kts
taboolib{
    relocate("top.maplex.arim","xxx.xxx.arim")
}
```
